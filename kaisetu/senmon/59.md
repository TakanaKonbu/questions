# 問1：正解3

## 1. 3つの重要キーワードをマスターしよう

この問題を解くために、気象庁がデータを集めるときの「基本のイメージ」を日常の言葉に翻訳してみましょう。

### ① 海面気圧 (a)

- **日常のイメージ：** **「全員、同じスタートライン（海の高さ）に揃えたときの空気の重さ」**
- **ルール：** 山の上と海の上では、頭の上にある空気の量が違うので、そのままでは天気の比較ができません。そのため、山の上の気圧を「もしここが海抜0mだったら…」と計算で直す必要があります。

### ② 気温の観測値 (b)

- **日常のイメージ：** **「今、その場所で生活している人が肌で感じる温度」**
- **ルール：** 天気予報の「今日の最高気温」は、その街の実際の高さで測った温度じゃないと意味がありません。海抜0mの高さにわざわざ直して発表したら、山の上の街の人は大混乱してしまいます。

### ③ 風速の観測値 (c)

- **日常のイメージ：** **「ビルや木に邪魔されていない、その場所のありのままの風の強さ」**
- **ルール：** 風は地面に近いほど、建物や地面の摩擦（抵抗）で弱くなります。そのため、なるべく障害物の影響を受けない高い場所（建物の屋上など）に風速計を設置して、**そこで測った値をそのまま使います。**

## 2. (a)〜(c)をルールに当てはめて判定！

それでは、問題文の記述が正しいかどうか、1つずつチェックしていきましょう。

### (a) 海面気圧の求め方

- **問題文の要約：** 観測した気圧や気温、高さを「物理の公式（換算式）」に当てはめて、海面の高さの気圧を計算で出している。
- **判定：** **正（〇）**
- **解説：** 日本全国の気圧を比べるために、標高が違う場所のデータを「海抜0m（海面）」の高さに統一する計算（海面更正）を行います。このとき、空気のバランスの式（静力学平衡の式）などを使って正しく計算しています。文の通り大正解です。

### (b) 気温の観測値

- **問題文の要約：** データのバラつきをなくすために、測った気温をわざわざ「海面の高さの気温」に計算し直して観測値にしている。
- **判定：** **誤（×）**
- **解説：** もしこれをやってしまうと、例えば標高が約1000mある長野県軽井沢町の気温が、天気予報では「東京と同じくらい暑い気温」に補正されて表示されることになってしまいます。気温は「その場所（標高）で実際に観測された値」をそのまま使います。

### (c) 風速の観測値

- **問題文の要約：** 高い所で測った風速は、地面の摩擦を計算に入れて「地上10mの高さの風速」に引き下げて計算し直している。
- **判定：** **誤（×）**
- **解説：** 風速計は、周りの建物などの影響を避けるために高い場所に置かれますが、そこで「実際に測った値」をそのままその地点の風速とします。わざわざ数式を使って地上10mの風速へと逆算するようなことはしません。

## 3. まとめ

3つの判定を並べてみましょう。

- (a) **正**（気圧はみんなで比べるために、海の高さに計算し直す）
- (b) **誤**（気温はそのままじゃないと、生活の役に立たない）
- (c) **誤**（風速は高い所で測った値を、そのまま使う）

「正・誤・誤」となっている選択肢は **③** になります！

> [!TIP]
> 
> 
> **試験に役立つ覚え方**
>

> 
> - **気圧：** 周りと比べるのが大事だから、計算して**高さを揃える！**
> - **気温・風：** その場所のリアルが大事だから、**いじらずそのまま使う！**
> 
> この区別がついていれば、この手の問題は一発で仕留められますよ！

# 問2：正解5

## 1. 2つのルール（色の意味と渦の向き）を頭に入れよう

この問題を解くカギは、図に書かれている「色」と「渦」のルールをシンプルに整理することです。

### ① 色のルール（ドップラー速度）

レーダーから電波をビームのように飛ばして、雨粒がどちらに動いているかを見ています。

- **暖色系（茶・赤）：** レーダーから「遠ざかる」動き（バイバイの赤）
- **寒色系（青）：** レーダーに「近づく」動き（こっちにくる青）

### ② 渦のルール（低気圧性の渦）

- 天気予報の台風などを思い出してください。北半球の「低気圧性の渦（メソサイクロンなど）」は、中心に向かって「反時計回り（左回り）」にグルグル回転します。

## 2. 「レーダーの視点」になってシミュレーションしよう

左下のパラボラアンテナマーク（レーダー）の場所に自分が立っていると想像してください。そこから斜め右上（北東方向）にある「反時計回りの渦」をジッと見つめてみます。

1. **渦の「左側」を見てみる：**
    
    反時計回りに回っているので、渦の左側にある空気は、ぐるっと回って「自分（レーダー）のほうへ向かってくる」動きになります。
    
    - 自分に近づいてくるので、色は「青色（寒色系）」になるはずです。
2. **渦の「右側」を見てみる：**
    
    逆に渦の右側にある空気は、ぐるっと回って「自分（レーダー）から奥へ遠ざかる」動きになります。
    
    - 自分から離れていくので、色は「茶・赤色（暖色系）」になるはずです。

つまり、レーダーから見て「左が青、右が赤」になっている図を探せばいいのです！

## 3. 選択肢を1つずつチェック！

この「左が青、右が赤」の目線で①〜⑤の選択肢をチェックしてみましょう。

- **①・②：** 青と赤が「手前と奥（縦方向）」に並んでいます。これは回転ではなく、風が一方向から吹き付けたり（収束）、逆に広がったり（発散）している状態なので×です。
- **③：** 4つも色が点在していて、複雑すぎるので×です。
- **④：** 「左が赤、右が青」になっています。これは反時計回りの逆、つまり「時計回り（高気圧性の渦）」の動きを表しているので×です。
- **⑤：** 見事に「左が青（近づく）、右が赤（遠ざかる）」になっています！これこそが、反時計回りに回る低気圧性の渦の証拠です。

したがって、正解は **⑤** になります。

## 4. まとめ

判定のエッセンスをスッキリまとめておきますね。

- **暖色（赤）** ＝ 遠ざかる
- **寒色（青）** ＝ 近づく
- **低気圧の渦** ＝ 反時計回り ➔ レーダーから見て「左が青、右が赤」になる！

> 
> その矢印に対して、反時計回りの矢印をぐるっと描いてあげると、「こっちは近づく（青）」「あっちは遠ざかる（赤）」が視覚的にパッと一瞬で判断できるようになりますよ！

# 問3：正解2

## 1. 3つの重要キーワードをマスターしよう

この問題を解くために、ウィンドプロファイラと前線の仕組みを日常の言葉に翻訳してみましょう。

### ① ウィンドプロファイラの仕組み (a)

- **日常のイメージ：** **「上空の5つの空に電波を投げて、跳ね返りで風を測る」**
- **ルール：** 真上、東、西、南、北の「5方向」に電波を打ち上げます。風に流されている空気の乱れに電波が当たって戻ってくると、救急車が通り過ぎるときに音が変わる（ドップラー効果）のと同じ現象が起きます。この「周波数のズレ」を計算して、上空の風の向きと強さを突き止めます。

### ② 温暖前線の形 (b)

- **日常のイメージ：** **「冷たい空気のすべり台を、温かい空がはい上がっていく形」**
- **ルール：** 温暖前線は、冷たい空気（重い）の上に、温かい空気（軽い）が「ゆるやかな坂道」のようによじ登っていく構造をしています。

### ③ 雨が降ったときの電波 (c)

- **日常のイメージ：** **「透明な空気の乱れより、目に見える雨粒のほうが電波を強く跳ね返す」**
- **ルール：** 普段は空気のわずかな乱れをキャッチしていますが、雨が降ると、電波は巨大なターゲットである「雨粒」に思いきりぶつかって大反射します。

## 2. (a)〜(c)をルールに当てはめて判定！

それでは、各文章をチェックしていきましょう。

### (a) ウィンドプロファイラの観測方法

- **問題文の要約：** 5方向へ電波を出し、跳ね返ってきた電波の周波数のズレから上空の風を測る。
- **判定：** **正（〇）**
- **解説：** まさにウィンドプロファイラの教科書通りの説明です。真上＋前後左右の「5方向」に電波を出すことで、立体的な風の動きを正確にキャッチしています。

### (b) 温暖前線が通過するときの見え方

- **問題文の要約：** 温暖前線が通るとき、まず「地面近く」から風が変わり、時間が経つにつれてその変化が「上空」へと厚くなっていく。
- **判定：** **誤（×）**
- **解説：** ここがこの問題最大の引っ掛けポイントです！
    
    温暖前線は「冷たい空気のすべり台を温かい空気が登る」形をしているため、前線が近づいてくるとき、**まず最初に「上空」から温かい南風に切り替わります。** その後、時間が経つにつれて、南風のエリアがどんどん下（地面付近）へと降りてくるのです。
    
    文では「地表付近から始まり、上空へ厚くなる」と逆のことが書かれているため誤りです。
    

### (c) 雨が降っている場合のデータ

- **問題文の要約：** 雨の日は、空気の乱れよりも雨粒による電波の跳ね返りの方が強いので、データは雨粒の動きになる。
- **判定：** **正（〇）**
- **解説：** 正解です！雨粒のほうが電波を強く跳ね返すため、雨の日は「雨粒が風に流されながら落ちていく動き」を捉えることになります。ただ、雨粒は風に乗って横移動するので、これを利用して高度ごとの風を計算することができます（垂直方向は雨の落下速度が加算されます）。

## 3. まとめ

判定を並べると以下の通りです。

- (a) **正**（5方向の電波のズレで風を測る）
- (b) **誤**（温暖前線は、上空から先に風が変わる）
- (c) **正**（雨の日は雨粒のほうが目立つので、雨粒の動きを測る）

これに一致するのは、選択肢 **②** です。

> [!IMPORTANT]
> 
> 
> **試験に役立つ覚え方（前線の形）**
> 
> 前線の問題が出たら、頭の中で**「坂道の形」**をイメージしましょう！
> 
> - **温暖前線：** ゆるやかな坂道がやってくるので、**まず「高い上空」からぶつかります。**（だから上空から風が変わる➔(b)はバツ！）
> - **寒冷前線：** 絶壁のような急坂（ドーム状）がやってくるので、**「地面近く」から一気にぶつかります。**
> 
> この「どっちから先にぶつかるか」のイメージを持っておくと、前線の通過問題で絶対に迷わなくなりますよ！

### ② 線形と非線形（カルマンフィルタ vs ニューラルネットワーク） (b)

- **日常のイメージ：** * 線形 ➔ 「気温が1度上がれば、売れるアイスが10個増える」という**単純な直線関係**。
    - 非線形 ➔ 「気温、湿度、風が複雑に絡み合って大雨が降る」という**複雑な曲線の関係**。
- **ルール：** カルマンフィルタは「直線的な関係（線形）」を直すのが得意です。一方、人間の脳を模したニューラルネットワーク（AIの仲間）は、直線では表せない「複雑な関係（非線形）」を見つけるのが得意です。

### ③ 系統誤差 (c)

- **日常のイメージ：** **「この体重計、いつも2キロ多めに表示されるな」という決まったクセ**
- **ルール：** スーパーコンピューターの計算にも、「昼の気温は高めに出やすい」「夜の冷え込みを弱めに見積もりやすい」といった時間帯による特有のクセ（系統誤差）があります。ガイダンスは、この決まったクセを引き算して直す役割を持っています。

## 2. (a)〜(c)をルールに当てはめて判定！

それでは、各文章の正誤をチェックしていきましょう。

### (a) カルマンフィルタの出力値

- **問題文の要約：** 入力する数値予報の結果（例えば、明日の予想気温15℃）が同じなら、計算を始めた初期時刻が違っていても、ガイダンスが出す最終的な答えはいつも同じになる。
- **判定：** **誤（×）**
- **解説：** カルマンフィルタは、初期時刻（例えば「朝9時スタートのデータ」と「夜21時スタートのデータ」）ごとに、**それぞれ独立して過去のズレを学習し、修正用パラメーターを日々更新しています。** そのため、たまたま同じ「15℃」という入力値であっても、初期時刻が違えば適用される修正用パラメーターが異なるため、**最終的な出力値（答え）は異なる値になります。** 「常に同じ値になる」という文は誤りです。

### (b) カルマンフィルタとニューラルネットワークの違い

- **問題文の要約：** カルマンフィルタは「線形（直線関係）」に使い、ニューラルネットワークは「非線形（複雑な関係）」にも使える。
- **判定：** **正（〇）**
- **解説：** 文の通り大正解です。カルマンフィルタはすっきりした直線的な誤差修正が得意で、ニューラルネットワーク（AI技術）は「降水確率」や「発雷確率」のように、たくさんの要素が複雑に絡み合う現象の予報によく使われます。

### (c) 系統誤差の低減

- **問題文の要約：** 数値予報には昼と夜で誤差の傾向（クセ）が変わることがあるが、ガイダンスはそれを減らすことができる。
- **判定：** **正（〇）**
- **解説：** これこそがガイダンスの一番の存在意義です！「昼は高めに出る」「夜は低めに出る」というそれぞれのクセを過去のデータから学習し、きれいに補正して正しい天気予報に直してくれます。

## 3. まとめ

判定を並べると以下の通りです。

- (a) **誤**（初期時刻が違えば、学習内容が違うので答えも変わる）
- (b) **正**（カルマン＝線形、ニューラルネットワーク＝非線形もいける）
- (c) **正**（昼夜の予報のクセを直すのがガイダンスの仕事）

これに一致するのは、選択肢 **④** ですね。

- **ルール：** スーパーコンピューターは「昨日の予報」をベースにしたデータ（第一推定値）を持っています。そこに、今日実際にアメダスや衛星が測った「本物のデータ」を上手にブレンドして、最も正しい現在の天気（初期値）を作り出します。

### ② ハイブリッドデータ同化手法 (b)

- **日常のイメージ：** **「これまでのデータ同化の技術に、さらに『予報のバラつき（アンサンブル）』の知恵を合体させた最新ハイテク手法」**

- **日常のイメージ：** **「じっくり時間をかけて全員の提出を待つか、スピード優先でそこにある分だけで計算を始めるか」**
- **ルール：** 世界中を予測する「全球モデル」は時間がかかるので、世界中からのデータが集まるのをじっくり待ちます。逆に、数時間後の大雨を予測する「局地モデル」はスピードが命なので、集まりの悪い海外のデータなどは待たずに、すぐに届いたデータだけで計算を開始します。

### ④ 3次元と4次元変分法 (d)

- **日常のイメージ：** * 3次元変分法 ➔ 「ある一瞬のストップモーション写真」を合わせていく方法。
    - 4次元変分法 ➔ 「時間の流れ（動画）」をそのまま意識して、過去から現在への動きのズレを滑らかに合わせていく方法。

## 2. (a)〜(d)をルールに当てはめて判定！

- **解説：** **ここが今回の間違い（引っ掛け）です！**
    
    土砂災害や洪水の危険を「今降っている雨」だけで判定すると、気づいたときには手遅れになってしまいます。そのため、キキクルでは最大で**数時間先（土砂災害や洪水では数時間先、浸水害では1時間先）までの「雨量予測値」を計算に組み込んで先回りして色を変えています。**「予測値は用いず」という記述は誤りです。
    

## 3. まとめ

3つの下線部の判定を並べると以下の通りです。

- (a) **正**（5段階の色分けで、指数と災害基準を使っている）
- (b) **正**（紫は全員避難の警戒レベル4に相当する）
- (c) **誤**（今だけのデータではなく、数時間先の予測値もしっかり使っている）

これに一致するのは、選択肢 **②** になります！

>

> 防災のためのシステムなので、「今現在」のデータで動くアメダス実況などとは違い、「未来の予測値（これから降る雨）」を貪欲に取り込んで動いている**というのが、試験で一番命になる知識ですよ！

# 問14：正解2

## 1. 3つの計算ルールをマスターしよう

予報の評価で用いられる3つの指標を算出するための定義と計算式です。

- **適中率：** 予報と実況が一致した日数（「予報◯・実況◯」および「予報●・実況●」）の合計を、全体のトータル日数（7日間）で割った値。
- **スレットスコア：** 「予報か実況の少なくともどちらか片方でも●（降水あり）であった日数」を分母とし、そのうち「予報と実況がともに●（降水あり）で一致した日数」を分子として割った値。
- **ブライアスコア：** 確率予報の誤差を評価する指標。各日の「予報確率（0〜1の数値）」と「実況（降水あり●なら1、降水なし◯なら0）」の差を2乗し、それを全日数分足し合わせて、全体のトータル日数（7日間）で割った平均値（値が小さいほど精度が高い）。

## 2. 観測データをルールに当てはめて判定！

予報区Aと予報区Bのデータをそれぞれ正確に数え上げて計算します。

### (a) 適中率の比較

予報と実況が完全に一致している日を数えます。

- **予報区Aの一致した日：**
    - 1日目（◯、◯）
    - 4日目（●、●）
    - 5日目（●、●）
    - 7日目（●、●） ➔ **合計 4日間**
    - 適中率 ＝ $\frac{4}{7}$
- **予報区Bの一致した日：**
    - 1日目（◯、◯）
    - 2日目（◯、◯）
    - 4日目（●、●）
    - 5日目（●、●）
    - 7日目（◯、◯） ➔ **合計 5日間**
    - 適中率 ＝ $\frac{5}{7}$
- **判定：** 適中率は予報区B（$\frac{5}{7}$）の方が予報区A（$\frac{4}{7}$）より高いため、文章(a)は「正」となります。

### (b) 降水ありの予報のスレットスコアの比較

「予報か実況の少なくともどちらかが●（降水あり）」の日を分母、ともに●の日を分子にします。

- **予報区A：**
    - 雨（●）が絡んだ日：2日目、3日目、4日目、5日目、6日目、7日目 ➔ **合計 6日間**（分母）
    - 予報・実況ともに●の日：4日目、5日目、7日目 ➔ **合計 3日間**（分子）
    - スレットスコア ＝ $\frac{3}{6} = 0.5$
- **予報区B：**
    - 雨（●）が絡んだ日：3日目、4日目、5日目、6日目 ➔ **合計 4日間**（分母）
    - 予報・実況ともに●の日：4日目、5日目 ➔ **合計 2日間**（分子）
    - スレットスコア ＝ $\frac{2}{4} = 0.5$
- **判定：** スレットスコアは予報区A、予報区Bともに 0.5 で同じであるため、文章(b)は「正」となります。

### (c) 降水確率予報のブライアスコアの比較

各日の「（予報確率 － 実況）の2乗」を算出し、その7日間の合計（平均をとる前の総合ペナルティ値）を比較します。

※確率は 0%＝0、50%＝0.5、100%＝1 として計算。実況は ●＝1、◯＝0。

- **予報区A：**
    - 1日目：$(0 - 0)^2 = 0$
    - 2日目：$(0 - 1)^2 = 1.0$
    - 3日目：$(0.5 - 0)^2 = 0.25$
    - 4日目：$(1.0 - 1)^2 = 0$
    - 5日目：$(1.0 - 1)^2 = 0$
    - 6日目：$(0.5 - 0)^2 = 0.25$
    - 7日目：$(1.0 - 1)^2 = 0$
    - 2乗誤差の合計 ＝ $0 + 1.0 + 0.25 + 0 + 0 + 0.25 + 0 = \mathbf{1.5}$
- **予報区B：**
    - 1日目：$(0 - 0)^2 = 0$
    - 2日目：$(0 - 0)^2 = 0$
    - 3日目：$(0 - 1)^2 = 1.0$
    - 4日目：$(0.5 - 1)^2 = 0.25$
    - 5日目：$(1.0 - 1)^2 = 0$
    - 6日目：$(0.5 - 0)^2 = 0.25$
    - 7日目：$(0 - 0)^2 = 0$
    - 2乗誤差の合計 ＝ $0 + 0 + 1.0 + 0.25 + 0 + 0.25 + 0 = \mathbf{1.5}$
- **判定：** 両予報区ともに2乗誤差の合計が 1.5 で等しく、ブライアスコア（それを7で割った値）も同じになるため、予報精度に差はありません。「予報区Bの方が予報精度が高い」とする文章(c)は「誤」となります。

## 3. まとめ

各文章の正誤の組み合わせは以下の通りです。

- (a) **正**（適中率はBの方が高い）
- (b) **正**（スレットスコアは双方同じ 0.5）
- (c) **誤**（ブライアスコアは等しいため、Bの方が精度が高いわけではない）

これに一致する選択肢は **②** になります。

# 問15：正解5

## 1. 2つの重要キーワードをマスターしよう

まずは、グラフや図を読み解くための「道具（知識）」を日常の言葉に翻訳しておきましょう。

### ① 外向き長波放射量（OLR） (a), (b)

- **日常のイメージ：** **「地球から宇宙へ逃げていく熱（赤外線）の量」**
- **ルール：** * **グラフが低い（数字が小さい）** ➔ 背の高いモクモクした雲（対流雲）に熱が遮られているということです。つまり、**「雨雲が多くて、対流活動が活発（天気が悪い）」**。
    - **グラフが高い（数字が大きい）** ➔ 邪魔する雲がなく、地面や海の熱がそのまま宇宙へ抜けています。つまり、**「晴れていて、対流活動が不活発（天気がいい）」**。

### ② フィリピンの天気と日本の夏の連動（PJパターン） (c), (d)

- **日常のイメージ：** **「フィリピンでお祭り騒ぎ（大雨）があると、日本もつられて超お祭り騒ぎ（猛暑）になる」という遠隔連動システム**。
- **ルール：** * フィリピン付近の対流活動が**活発** ➔ 日本付近の太平洋高気圧が**強くなり、猛暑・晴天**になる。
    - フィリピン付近の対流活動が**不活発** ➔ 日本付近の太平洋高気圧が弱くなり、冷夏・長雨（曇りや雨が多い）になる。

## 2. 空欄 (a)〜(d) を順番に埋めていこう！

それでは、お送りいただいた解説のステップを実況中継しながら、パズルを完成させていきましょう。
