# 解答と解説

#### 問1 【正解】1

- **解説**
    - (a) **正**：赤外差分水蒸気画像（ダスト画像など）は、エーロゾルの赤外線吸収特性の差を利用して夜間でも黄砂の領域を明瞭に判別できます。
    - (b) **正**：可視画像は太陽光反射を利用するため夜間は使えませんが、空間解像度が赤外画像より高いため、霧や積乱雲の微細構造の把握に有効です。
    - (c) **正**：水蒸気画像は、雲がない領域でも対流圏中・上層の水蒸気の多寡による放射を捉えるため、暗化（ダークスロット）からジェット気流や気圧の谷などの大気循環を把握できます。
    - (d) **誤**：赤外画像で輝度温度が非常に低い（白い）領域は、上空高くに達する雲を意味しますが、これが巻雲などの薄い上層雲である場合は激しい雨は降りません。また、発達した積乱雲であっても必ずその周囲全体で激しい雨が降っているとは限りません。

    - (c) **誤**：視線速度は、レーダーに「向かう（接近）」または「遠ざかる（離れ）」方向の風速成分のことです。同心円の接線方向（直交する方向）の風速成分は測ることができません（ゼロになります）。

#### 問4 【正解】1

- **解説**
    - (a) **正**：データ同化は、第一推定値（予測値）と実際の観測データを統計的・力学的な整合性を考慮して融合し、最適な初期値を作る技術です。
    - (b) **正**：観測値をそのまま格子点に代入すると、風と気圧のバランス（地衡風バランスなど）が崩れ、非物理的な高周波重力波ノイズが立ち上がり、計算が破綻することがあります。これを防ぐのが初期値化です。

    - (d) **正**：アンサンブルデータ同化の最大の特徴は、複数の予報メンバーのばらつきから「その時々の気象状況に応じた誤差（背景誤差共分散）」を動的に推定してデータ同化に利用できる点です。

#### 問7 【正解】1

#### 問15 【正解】3

- **解説**
    - **スレットスコア（適中率：TS）の計算公式**
        
        スレットスコアは、予報と実況の少なくともいずれかで「現象（降水あり）の発生」があった回数を分母とし、予報と実況がともに「降水あり」で一致した回数を分子とする指標です（現象が起きなかった「予報×・実況×」のケースを評価から除外する）。
        
        $TS = \frac{A}{A + B + C}$
        
        - A（予報○・実況○：適中）：1日、4日、8日、10日の計 **4回**
        - B（予報○・実況×：空振り）：2日、7日の計 **2回**
        - C（予報×・実況○：見逃し）：5日の計 **1回**
        - D（予報×・実況×：なしで適中）：3日、6日、9日の計 3回（これは計算に含めない）
    - **計算の実行**
        
        分母（現象に関わった日数）$= A + B + C = 4 + 2 + 1 = 7$
        
        分子（適中日数）= A = 4
        
        $TS = \frac{4}{7} \approx 0.5714...$
        
        したがって、最も適切な選択肢は **3** （0.57）となります。
